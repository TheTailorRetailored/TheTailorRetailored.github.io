# cv.lachlanbridges.com

The source for my portfolio: selected software projects, applied mathematics
background and contact details.

The site is deliberately plain HTML and CSS. It has no runtime dependencies,
analytics, cookies or third-party assets. GitHub Actions checks local links and
publishes the site with GitHub Pages.

## Local preview

```bash
python scripts/check_site.py
python -m http.server 4173
```

Then open <http://localhost:4173>.
